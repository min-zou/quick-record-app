Quick Record 隐私政策

更新日期：2026年5月5日

Quick Record 是一款本地优先的快速文字记录工具。我们高度重视您的隐私，本政策说明我们如何收集、使用和保护您的信息。

1. 数据存储位置

您的所有文字记录默认存储在设备本地的 IndexedDB 中，不会自动上传到任何服务器。

只有在您主动点击"同步"按钮时，应用才会通过 GitHub API 将记录同步到您自己配置的私有 GitHub 仓库。

2. GitHub Token 安全

GitHub 访问令牌（Token）仅保存在您当前设备的浏览器本地存储（localStorage）中：

  • 不会传输到除 GitHub API 之外的任何第三方服务
  • 不会存储在我们的服务器上（我们没有任何服务器）
  • 您随时可以清除浏览器数据来删除该令牌

3. 收集的信息

我们不收集任何个人身份信息。应用仅存储：

  • 您输入的文字记录内容
  • 记录创建时间戳
  • 设备标识符（用于区分多设备来源）

4. 第三方服务

本应用仅与 GitHub 进行交互（当您配置同步后）。您的数据通过 GitHub 的服务条款和隐私政策进行管理。

5. 离线使用

Quick Record 支持完全离线使用。没有网络连接时，您可以正常保存和查看记录，待网络恢复后再手动同步。

6. 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系我们。

本政策可能随应用更新而调整，重大变更将在应用内提示。
